Getting Started

Learn how to use the NVIDIA Component Hub to build beautiful, consistent websites that follow NVIDIA's design system.

Introduction

The NVIDIA Component Hub is a comprehensive collection of reusable components and sections designed specifically for NVIDIA websites. It provides a consistent design system that ensures all NVIDIA web properties maintain the same look, feel, and functionality.

Note: This hub is designed for developers who need to create NVIDIA-styled websites quickly and consistently. All components follow NVIDIA's official design guidelines and CSS framework.

Quick Start

Get up and running with the NVIDIA Component Hub in just a few steps:

1Choose a Template

Start with one of our pre-built templates: Project Template, Lab Template, or Project List. These provide a solid foundation for your specific use case.

2Browse Components

Explore the component library to find the sections you need. Each component includes live examples and copyable code.

3Copy and Customize

Copy the HTML code for your chosen components and customize the content to match your project's needs.

Basic Template Structure

Every NVIDIA website should include these essential elements:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Page Title</title>
    
    <!-- NVIDIA CSS Dependencies -->
    <link rel="stylesheet" href="https://nvidia.com/assets/base/clientlibs/main.css">
    <link rel="stylesheet" href="src/styles/nvidia-components.css">
</head>
<body>
    <!-- NVIDIA Navigation -->
    <nav style="height: 44px; display: flex; align-items: center; background-color: var(--color-gray-0); border-bottom: 1px solid var(--color-gray-200);">
        <div class="nv-container" style="width:1200px; display: flex; align-items: center;">
            <!-- NVIDIA Logo SVG -->
        </div>
    </nav>
    
    <!-- Your Content Here -->
    
    <!-- NVIDIA Footer -->
    <footer class="nv-footer">
        <!-- Footer content -->
    </footer>
</body>
</html>

Page Navigation

The hub's navigation is organized into logical groups to help you find components quickly:

Header Section

  • Header Banner: Main page header with breadcrumbs, title, and description
  • Intro Section: Hero section with project information and call-to-action buttons
  • Meet the Team: Team member showcase with photos and bios

Text Section

  • Bullet List Section: Styled unordered lists for key points
  • Table Section: Data tables with NVIDIA styling
  • Code Container: Syntax-highlighted code blocks

Teaser Section

  • Feature Teasers: 2 and 3-column feature showcases
  • Media Teasers: 3 and 4-column media card layouts
  • Image Sections: Single, double, and triple image layouts

Miscellaneous

  • Before/After Image Slider: Interactive image comparison tool
  • Border Section: Simple separator lines
  • Color Section: Color palette demonstrations

Templates

  • Project Template: Complete research project page with Before/After slider
  • Lab Template: Research lab overview page
  • Project List: Portfolio-style project listing
Pro Tip: Use the navigation's scrollbar on smaller screens to access all components. The navigation is sticky and will stay visible as you scroll through the content.

CSS Dependencies

To ensure your components display correctly, include these CSS files in your HTML:

<!-- NVIDIA Base Client Libraries -->
<link rel="stylesheet" href="https://nvidia.com/assets/base/clientlibs/main.css">
<link rel="stylesheet" href="https://stage.nvidia.com/assets/base/clientlibs/main.css">
<link rel="stylesheet" href="https://nvidia.com/assets/base/clientlibs/main.css">

<!-- NVIDIA Component Hub Styles -->
<link rel="stylesheet" href="src/styles/nvidia-components.css">

CSS Classes Overview

The hub uses several key CSS classes:

  • .nv-section - Main section container with padding
  • .nv-container - Content wrapper with max-width and centering
  • .nv-h--large - Large heading style
  • .nv-h--medium - Medium heading style
  • .nv-h--small - Small heading style
  • .nv-text-large - Large body text
  • .nv-text-medium - Medium body text
  • .nv-cta-primary - Primary call-to-action button
  • .nv-cta-secondary - Secondary call-to-action button

Best Practices

Follow these guidelines to create consistent, professional NVIDIA websites:

Content Structure

  • Start with a Header Banner for context and navigation
  • Use Intro Section for main project information
  • Organize content with clear section headings
  • Include relevant call-to-action buttons
  • End with appropriate footer information

Design Consistency

  • Use NVIDIA's color palette (green: #76b900)
  • Maintain consistent typography hierarchy
  • Follow spacing guidelines (mb-4, mb-5, etc.)
  • Ensure responsive design on all devices

Accessibility

  • Include proper alt text for images
  • Use semantic HTML elements
  • Ensure sufficient color contrast
  • Provide keyboard navigation support

Performance

  • Optimize images for web use
  • Minimize custom CSS when possible
  • Use CDN links for external resources
  • Test loading times on various devices
